Bienvenue sur notre site réalisé par les étudiants MMI du groupe B2 de l’IUT de Lens. Vous avez l’habitude de travailler en équipe et désirez une plateforme de communication interactive, le système Git est fait pour vous ! Nous allons vous aider à l’aide de tutoriels à vous familiariser avec celui-ci.

Tout d’abord l’intégrateur va créer un répertoire sur GitHub . Une fois terminé les autres membres de l’équipe doivent aller sur le profil GitHub de l’intégrateur. Aller dans « Repositories » puis sur le nom du répertoire (exemple : TutoGit, TP2). Ensuite, il faut en haut à droite cliquer sur « Fork ». Pour l’intégrateur ne pas oublier de cliquer sur « read me ».
